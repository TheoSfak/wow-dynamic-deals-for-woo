== Installation ==

1. Upload the plugin files to `/wp-content/plugins/woo-dynamic-deals/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Dynamic Deals
4. Enable desired modules in Settings tab
5. Create your first rules

== Frequently Asked Questions ==

= Can I combine multiple rules? =

Yes, rules execute based on priority. Use "Stop further rules" to prevent cascade.

= Do price rules work with sale prices? =

Yes, choose "Apply To" option: regular, sale, or both prices.

= Can I schedule rules for specific days? =

Yes, use the scheduling section to set dates, times, and days of week.

= How do I target wholesale customers? =

Use user role restrictions to target specific roles like "Wholesale Customer".

= Is HPOS compatible? =

Yes, fully compatible with High-Performance Order Storage.
